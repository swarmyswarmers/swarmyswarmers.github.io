About Swarmy Swarmers

This project involves building and programming remote controlled cars that act together as a swarm. More specifically this requires at least two cars (preferably three) that follow a self-elected leader. The leader car will be remotely controlled by a user and send follower commands to following cars. If the leader car loses connection with the other cars or the leader loses connection with the user, then the remaining cars will self-elect a new leader car (using a RAFT-like election protocol) and reconnect to the user to receive directions. Additionally, all cars will be constantly running collision avoidance software, this removes the need to do grid-based positioning and is more scalable.

This project was built for our final term project for CS1600 (Real-Time and Embedded Software) Course.
